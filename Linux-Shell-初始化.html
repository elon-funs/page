Linux-Shell-初始化

andy发表:2020-01-01 22:22:22

#>0.思路
  拿到一台新机如何把命令行趁手工具快速初始化呢？
  先把命令行补全,大小写忽略搞起来,
  常用工具安装好vim git等脚本化,
  把个人的常用alias,自定义命令/函数导入,此处推荐本人的一个库脚本

#>1.先把命令行补全,大小写忽略搞起来
    
set completion-ignore-case on
set show-all-if-ambiguous on
TAB: menu-complete


#>2.常用工具安装好vim git等脚本化
  
yum install git
yum install vim
yum install curl
yum install zsh
chsh -s /bin/zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"


#>3.本人的一个库脚本
  
git clone https://gitlab.com/elonfuns/autoshell.git

  php faker 这个库是国际化的大库 https://github.com/fzaninotto/Faker
  composer 安装就好了
  如果原库不能满足你的需求就加入自己的库，其实很简单
  在你调用的地方加入自己写的扩展文件
  $faker = new Faker\Generator();
  $faker->addProvider(new Faker\Provider\zh_CN\own($faker));
  $faker->teacher;

  own文件当然你归类着原有的抄即可，规则如下
  在Provider\zh_CN目录下编写
  一是空间与上面引用一致 命名空间对应的目录最好也保持一致
namespace Faker\Provider\zh_CN;

class Food extends \Faker\Provider\Food
{
    protected static $foods = ['面条','包子','大米'];

    public static function mainfood()
    {
        return static::randomElement(static::$foods);
    }

}

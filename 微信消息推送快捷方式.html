          原理就是公众号的消息推送。
          1.Server酱　建了公众号叫《方糖》，你关注然后接收此公众号消息推送
          2.Server酱还建有一个服务器，地址：https://sct.ftqq.com/，把你想发的消息推送到他的服务器上。
          3.Server酱把你想发的消息下发到你的微信号上。
          简不简单，方不方便！
          第一步，
            打开https://sct.ftqq.com/官网，用你想收消息的微信号来扫码登陆，
          第二步，
            登陆后会分配一个sendkey给你，并且在消息通道菜单中选择方糖服务号。（此处就是建立你推送到他服务器的消息，与接收的微信号之间的关联）
          第三步
            简单的curl 发送如下：
            https://sctapi.ftqq.com/$sendkey.send?title=$title&desp=$contentmsg

            函数封装方法如下:
            function sc_send(  $title , $desp = $contentmsg , $key = $SENDKEY'  )
            {
                $postdata = http_build_query( array( 'text' => $text, 'desp' => $desp ));
                $opts = array('http' =>
                array(
                    'method'  => 'POST',
                    'header'  => 'Content-type: application/x-www-form-urlencoded',
                    'content' => $postdata));
                
                $context  = stream_context_create($opts);
                return $result = file_get_contents('https://sctapi.ftqq.com/'.$key.'.send', false, $context);

            }

            其他查看手册，有查询推送状态、消息加密等方法
